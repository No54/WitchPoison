## 项目简介

"女巫的毒药"是一款多人桌面卡牌游戏，游戏开始时，玩家抽取具有不同技能的身份牌，然后每回合抽取食物牌，游戏中可以通过积分购买道具牌，需要运用策略和技能来获取积分或让其他玩家出局，最终成为游戏的胜利者。

## 游戏特色

- **多样化的身份系统**：包含多种不同身份，每个身份都有独特的技能和效果
- **丰富的食物牌**：多种食物牌，有的提供积分，有的则是致命的毒药
- **策略性道具**：各种道具牌，用于辅助自己或干扰其他玩家
- **完整的游戏流程**：从身份抽取到游戏结束的完整流程控制
- **调试模式**：便于开发和测试的调试功能

## 技术栈

- **Vue 3**：前端框架
- **Vite**：构建工具
- **JavaScript**：编程语言
- **CSS**：样式设计

## 游戏规则

### 游戏流程

1. **身份抽取**：玩家设置游戏人数并依次抽取身份牌
2. **游戏开始**：每位玩家获得初始3积分
3. **回合流程**：
    - 行动一阶段：使用身份技能或道具（如果有）
    - 行动二阶段：翻开一张食物牌，有的直接获得积分，有的具有特殊效果
    - 行动三阶段：可以使用金币购买道具牌，也可以不购买

4. **游戏结束**：抽到毒药牌时，玩家出局，当只剩一名玩家或毒药牌用完，游戏结束；若仅剩毒药牌，则积分最高者获胜，结算时道具牌按4积分计算。

### 身份类型

- 学者：游戏开始时，获得5点积分，而不是3点
- 魔术师：使用道具牌时，你可以查看桌上一张食物牌
- 少女：你可以跳过你的回合(一局游戏一次)
- 小女孩：你购买道具时只需花费4积分而不是5积分
- 守墓人：当你翻到“女巫的毒药”时，免出局一次，随后将它盖回去和其他食物牌重新洗混
- 女仆：游戏开始时，获得一张道具牌
- 老流氓：你可以获得一名其他玩家的3点积分(一局游戏一次)
- 长老：购买道具时，抽二保留一
- 纯白之女：可以将所有食物牌重新洗牌(一局游戏一次)
- 弓箭手：下一张食物牌的积分翻倍(一局游戏一次)

### 食物牌

- 女巫的毒药：抽到即出局，一共3张
- 苹果，获取1积分，一共6张
- 苹果派，获取2积分，一共3张
- 甜甜圈，获取3积分，一共2张
- 糖果，获取5积分，一共1张
- 棒棒糖，获得其他玩家的3积分，一共2张
- 曲奇饼干，下一张食物牌的积分翻倍，一共2张
- 放大镜，立即查看场上一张食物牌，一共2张
- 魔法草药，立即拿取一张道具牌，一共2张
- 能量饮料，立即再翻一张食物牌，一共2张

### 道具牌

- 窥视：查看3张食物牌
- 移形换影：选择2张食物牌偷偷交换位置
- 解毒剂：当你翻到”女巫的毒药“时，免出局一次，随后将它盖回去和其他食物牌重新洗混
- 强运甘露：摸取道具牌堆顶2张牌，然后选择1张加入手牌，另一张放回堆底
- 护身符：在你的下一个回合开始前，不会出局
- 力量药剂：选择一名其他玩家，你来帮他翻开一张食物牌，效果由他承担
- 点石成金：立即获取7点积分
- 妙手空空：获得一名其他玩家的3点积分

## 调试模式

游戏提供调试模式，便于开发和测试：

- 查看所有卡牌的内容
- 调整道具牌的顺序
- 快速测试各种游戏场景

在游戏界面点击"调试模式"按钮即可切换。
